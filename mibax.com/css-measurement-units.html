CSS Length Units

Unit Description
px Absolute unit, representing one pixel on the screen.
rem Relative to the font-size of the root element (usually the `` element). If the root font size is 16px, 1rem is equal to 16px.
% Percentage relative to the parent element. For example, if a parent element has a width of 200px, setting a child element's width to 50% would make it 100px.
em Relative to the font-size of the element itself. If the font size of the element is 16px, 1em is equal to 16px.
vw Equal to 1% of the viewport width. If the viewport width is 1000px, 1vw is equal to 10px.
vh Equal to 1% of the viewport height. If the viewport height is 800px, 1vh is equal to 8px.
vmin Equal to the smaller of vw or vh. It represents 1% of the smaller dimension of the viewport.
vmax Equal to the larger of vw or vh. It represents 1% of the larger dimension of the viewport.
ex Relative to the x-height of the current font. It is seldom used.
ch Relative to the width of the "0" (zero) glyph in the current font.
