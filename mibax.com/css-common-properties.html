Text Styling Properties

Property Sample Values Count Description
color #000000, #fff, #dc3545 3 Sets the text color or background color.
font-size 1.1rem, 3rem, 1.2rem, 0.85rem 4 Sets the size of the font in the element.
font-weight bold, 600 3 Sets the thickness of the font characters.
line-height 0.8, 1.2, 2.375rem, 80% 4 Sets the height of a line of text.
text-align center, left 5 Aligns text within an element.

Background Properties

Property Sample Values Count Description
background-color #fff 1 Sets the background color.
background-image url(...) 4 Sets the background image of an element.
background-repeat no-repeat 4 Specifies how a background image should be repeated.
background-size cover 4 Sets the size of the background image.
